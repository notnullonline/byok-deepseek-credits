# Change Log

## [0.0.1] - 2026-06-14

### Added
- DeepSeek API remaining balance shown in the VS Code status bar
- Secure API key storage via VS Code `SecretStorage` (never stored in `settings.json`)
- Auto-refresh on a configurable interval (default: 30 minutes)
- Commands: **Refresh Balances** and **Set API Key**
- Clicking the status bar item opens the DeepSeek usage page in the browser
